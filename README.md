# AdoFutur

> *Prendre de meilleures décisions pour son avenir*

## 📖 Présentation du projet

**AdoFutur** est une plateforme web d'orientation scolaire et
professionnelle conçue pour aider les élèves, étudiants et jeunes
diplômés à mieux préparer leur avenir académique et professionnel.

## Problématique

En Côte d'Ivoire, de nombreux jeunes choisissent leur filière ou leur
métier sans information claire sur : 
- les débouchés réels,
- lescompétences demandées,
- les possibilités d'insertion professionnelle,
- les perspectives salariales.

## Fonctionnalités principales

### Test d'orientation intelligent

Le test permet à l'utilisateur d'identifier les métiers adaptés selon: 
- sa personnalité,
- ses centres d'intérêt,
- ses compétences,
- son profil scolaire.

### Découverte des métiers

Chaque métier présente : 
- une description claire,
- les études nécessaires,
- le niveau requis,
- les débouchés,
- les compétences demandées.

### Calculateur de moyenne

Les utilisateurs peuvent calculer : 
- leur moyenne trimestrielle,
- leur moyenne annuelle,
- leur moyenne d'orientation.

### FAQ / Aide

Une section aide répond aux questions fréquentes : 
- orientation,
- métiers,
- calcul des moyennes,
- Recommandations de partenaire dans le but de s'améliorer,
- utilisation de la plateforme.

## Public cible

-   Élèves du secondaire

-   Étudiants
-   Parents
-   Conseillers d'orientation

## Objectif principal

Aider les jeunes à prendre de meilleures décisions pour leur avenir
